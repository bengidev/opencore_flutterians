# Security Policy

## Supported versions

This project is under active early development. Security fixes are applied on the `main` branch.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **bambang.trd17@gmail.com** with:

- A description of the issue
- Steps to reproduce (if possible)
- Impact assessment (if known)

You should receive an acknowledgment within a few days. Please give us a reasonable window to investigate and ship a fix before any public disclosure.
